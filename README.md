# AmritpalYadav-Portfolio

A premium Data Science Student portfolio built using React, Tailwind CSS v3, and Vite. Designed with an expert Aurora Dark Theme, glassmorphism layouts, full responsiveness, and custom interactive features.

## Features
- **Modern Aurora UI**: Blending deep midnight tones with animated glassmorphism gradients.
- **Dedicated Education Timeline**: Visual separation of academic journey.
- **Bento Box Certifications Grid**: Highlighted focus on continuous learning credentials.
- **Vite & React Ecosystem**: Extremely fast local setup with hot-module-replacement built-in.
